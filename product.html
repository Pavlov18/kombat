Танк охладитель молока открытого типа

Основным отличием установок для охлаждения молока открытого типа является применение медного многосекционного трубчатого испарителя, припаянного к внешней стороне днища ванны, по которому циркулирует хладагент (фреон R22). Данная конструкция, кроме высокого КПД и, следовательно, более эффективного охлаждения, имеет значительный запас прочности по повышенному давлению и не боится промывки ванны горячей водой. Сама ванна имеет удобную для обслуживания прямоугольную форму и изготовлена из пищевой нержавеющей стали, термоизолирована снаружи, облицована листовой нержавеющей сталью, имеет электромеханический перемешиватель с напряжением питания 12В и съемные верхние крышки. Температура охлажденного молока постоянно индицируется на пульте управления с цифровой индикацией. При достижении заданной температуры установка для охлаждения молока автоматически переходит в режим поддержания ее. Установки для охлаждения молока УОМ4000-УОМ6000 комплектуются элементами механической мойки. Все установки для охлаждения молока комплектуются широко распространенными компрессорно-конденсаторными агрегатами фирмы "Maneurop" и "L"Unite Hermetigue" (Франция).
